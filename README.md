# Awesome Proactive Agent

跟踪主动式智能体（Proactive Agent）领域的研究进展。

## 领域定义

主动式智能体不同于传统的被动响应式智能体——它能在用户未明确请求时，通过观察用户行为和环境上下文，主动推断意图并适时介入提供帮助。
